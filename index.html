HTML
Bahasa markup yang digunakan untuk membuat struktur dasar halaman web.
CSS
Digunakan untuk mengatur tampilan dan gaya halaman web, seperti warna, font, dan layout.
JavaScript
Bahasa pemrograman yang membuat halaman web menjadi interaktif dan dinamis.
Tailwind CSS
Framework CSS berbasis utility class yang memudahkan pembuatan desain responsif secara cepat dan konsisten.

Tutorial Membuat Website Sederhana

1. Siapkan Folder dan File
Buat satu folder baru, misalnya bernama websaya. Di dalamnya, buat tiga file utama: index.html, style.css, dan script.js.
2. Tulis Struktur Dasar HTML
Buka file index.html, lalu tulis struktur HTML dasar menggunakan tag <!doctype html>, <html>, <head>, dan <body>.
3. Tambahkan Gaya Menggunakan CSS
Di file style.css, tambahkan kode untuk mengatur tampilan halaman, seperti warna latar belakang, font, dan tata letak. Hubungkan file CSS ke HTML dengan tag <link rel="stylesheet" href="style.css">.
4. Tambahkan Interaktivitas dengan JavaScript
Gunakan file script.js untuk menambahkan logika dan interaksi, seperti tombol yang bisa diklik atau animasi sederhana. Hubungkan dengan tag <script src="script.js"></script>.
5. Gunakan Tailwind CSS (Opsional)
Tambahkan link CDN Tailwind di bagian <head> agar kamu bisa langsung memakai class Tailwind untuk styling cepat dan responsif. Misalnya: <script src="https://cdn.tailwindcss.com"></script>.
6. Jalankan di Browser
Buka file index.html di browser, dan lihat hasil websitemu. Kamu bisa menyempurnakannya dengan menambah konten, gambar, atau animasi.
